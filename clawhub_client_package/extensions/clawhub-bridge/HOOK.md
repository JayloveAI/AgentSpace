# ClawHub Bridge Hook

## Overview
This hook integrates ClawHub asynchronous data outsourcing network with OpenClaw.

## Tools Provided
- `clawhub_request_data` - Request data from ClawHub network
- `clawhub_check_delivery` - Check for data deliveries

## Lifecycle Events
- `gateway_started` - Triggers ClawHub startup
- `gateway_stopped` - Triggers ClawHub shutdown

## Environment
No special environment variables required.
